Multimedia

What is Multimedia?

Multimedia is a representation of information in an attractive and interactive manner with the use of a combination of text, audio, video, graphics and animation. In other words we can say that Multimedia is a computerized method of presenting information combining textual data, audio, visuals (video), graphics and animations.

The HTML5 Video

The Video Element

Used to show the video using this tag: <video>

The Video Element Attributes, Properties and Events:

Controls: attribute that adds video controls.
Width: sets the video width.
Height: sets the video height.
Autoplay: start the video automatically.
video.Paused: returns the current video state, true if pause, false if playing.

Sample codes using HTML5 video:

<video width="100%" height="auto" controls>
<source src="../files/Media/CrabRave.mp4" type="video/mp4">
<source src="../files/Media/CrabRave.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

The HTML5 Audio

The Audio Element

Specifies a standard way to embed audio in a web page using this tag: <audio>

The Audio Element Attributes, Properties and Events: Controls: attribute that adds audio controls.
Autoplay: start the audio automatically.
audio.Paused: returns the current audio state, true if pause, false if playing.

Sample codes using HTML5 audio:

<audio width="100%" controls>
<source src="../files/Media/CrabRave.mp3" type="video/mp3">
<source src="../files/Media/CrabRave.ogg" type="video/ogg">
Your browser does not support the audio tag.
</video>
